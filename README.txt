Serwer REST napisany w pythonie pod koniec stycznia

Serwer pozwala na przechowywanie notatek
Na notatke sklada sie indywidualne id przydzielane przez serwer, temat notatki, tresc notatki, tag i kategoria
Serwer wysyla i odczytuje pliki JSON
